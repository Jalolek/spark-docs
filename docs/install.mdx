<TabItem value="server" label="Minecraft Server" default>

To install spark, just need to download the latest spark **.jar** file from our [downloads page](https://spark.lucko.me/download) and place in the server `mods` or `plugins` folder.

spark will load the next time your server (re)starts.

You will be able to control spark with the `/spark` command.

You may need to grant yourself the `spark` permission, or ensure your user is a server operator first.

:::tip

You may also need to [perform some extra steps](misc/Using-async-profiler#system-requirements) for the `async-profiler` profiler engine mode to work correctly. 

:::

</TabItem>
<TabItem value="client" label="Minecraft Client">

You will be able to control spark with the `/sparkc` command.

</TabItem>
<TabItem value="proxy" label="Minecraft Proxy" default>

To install spark, just need to download the latest spark **.jar** file from our [downloads page](https://spark.lucko.me/download) and place in the proxy `plugin` folder.

spark will load the next time your proxy (re)starts.

You will be able to control spark with the `/sparkb` (BungeeCord) or `/sparkv` (Velocity) commands.

You may need to grant yourself the `spark` permission first.

:::tip

You may also need to [perform some extra steps](misc/Using-async-profiler#system-requirements) for the `async-profiler` profiler engine mode to work correctly. 

:::

</TabItem>
